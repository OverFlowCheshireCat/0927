경력 학술 인적 소개

자기 소개

안녕하십니까. 저는 소프트웨어 프로그래머를 꿈꾸고 있는 학생 김민철 입니다. 중학교 시절 처음으로 프로그래밍이라는 분야에 흥미를 가지게 되어, 처음으로 JAVA 라는 언어를 접해 보았으며, 그랬던 만큼 현재 가장 자신있고 잘하는 언어가 JAVA 라고 자부 할 수 있습니다. 그렇게 지금까지 독학을 진행해왔고, 대학교를 입학하며 프로그래밍을 전공하게 되었습니다. 어릴적 부터 경험해 온 덕에 다른 학생들보다 디버깅 능력이 우수하여 프로젝트 완성 속도가 빨라, 남은 시간에 프로젝트의 약점이나 모자란 부분들을 채워나가며 항상 프로젝트의 완성도를 높이고 사용자들의 요구사항을 최대한 고려한 프로그램들을 만들 수 있었습니다. 팀 프로젝트 등을 진행 시 항상 백 엔드 개발을 담당하고 있으며, 현재의 명확한 제 꿈은 소프트웨어 백 엔드 개발자 입니다. 앞으로 더 노력하여 유능한 개발자가 될 수 있도록 노력 할 것 입니다.
